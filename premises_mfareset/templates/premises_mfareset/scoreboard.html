Authorization model

Access is granted based on membership in Active Directory groups under: 

OU=MFAResetAdmins,OU=Groups,OU=SOC,OU=CIS,OU=AIT,DC=win,DC=dtu,DC=dk

Each group defines its scope using the attribute: extensionAttribute1.

Example: 

Group: win.dtu.dk/AIT/CIS/SOC/Groups/MFAResetAdmins/AIT
extensionAttribute1: AIT,STUDENTS

This grants permission to reset users under:

  • DTUBaseUsers/AIT
  • DTUBaseUsers/STUDENTS

If a user is a member of multiple groups, the allowed OUs are combined.
